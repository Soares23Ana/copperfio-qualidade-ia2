Sistema de Qualidade e Atendimento - Coperfil

Este repositório contém o código-fonte e a documentação do aplicativo Android desenvolvido para a empresa Coperfil, especializada na venda de cabos elétricos para empresas.

O sistema tem como objetivo centralizar atendimento, reclamações, catálogo de produtos e análise de feedbacks, oferecendo uma área exclusiva para clientes e um painel gerencial com dashboards estratégicos.

 Objetivo do Projeto

Desenvolver um aplicativo Android corporativo que permita:
	•	Melhorar o relacionamento com clientes
	•	Organizar reclamações e chamados
	•	Disponibilizar ficha técnica e catálogo de produtos
	•	Coletar avaliações e feedbacks
	•	Fornecer dados estratégicos para tomada de decisão da gestão

⸻

 Estrutura do Aplicativo

O aplicativo será dividido em duas áreas principais:

Área do Cliente

Funcionalidades disponíveis para usuários clientes:
	•	Tela de Login
	• Tela de Cadastro
	•	Recuperação de Senha
	•	Abertura de Reclamações e Chamados
	•	Consulta de Ficha Técnica dos Produtos
	• Catálogo de Produtos
	• Avaliação e Feedback de Produtos

Essa área permite que o cliente registre sua experiência e acompanhe o relacionamento com a empresa de forma digital e organizada.

⸻

Área do Gestor

Área restrita para usuários com perfil de gestão.

Funcionalidades:
	• Dashboard com gráficos de desempenho
	• Resumo geral das avaliações
	• Alertas de avaliações negativas
	• Visualização de chamados e reclamações
	• Monitoramento da satisfação do cliente

O objetivo dessa área é fornecer visão estratégica e apoiar decisões rápidas baseadas nos dados coletados no aplicativo.

⸻

Fluxo do Usuário
	1.	O cliente realiza login ou cadastro no aplicativo.
	2.	Acessa catálogo, ficha técnica ou registra uma reclamação.
	3.	Pode avaliar produtos e enviar feedback.
	4.	As informações são armazenadas no sistema.
	5.	O gestor acessa o dashboard para acompanhar métricas e alertas.

  Tecnologias Utilizadas
	•	Desenvolvimento Android
	•	Banco de Dados para armazenamento de usuários, avaliações e chamados
	•	Biblioteca de gráficos para dashboards
	•	Sistema de autenticação de usuários
